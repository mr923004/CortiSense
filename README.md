CortiSense

CortiSense is a telemetry-driven application for real-time cortisol monitoring via a wearable electrochemical sensor. Developed as part of a third-year undergraduate group project in the Department of Bioengineering at Imperial College London, the project is supervised by Professor Martyn Boutelle.

Project Summary

The system transmits continuous cortisol readings from a sensor to a secure cloud database using a client–server model. This data is accessible via two user interfaces:
* A mobile app for individuals to view live hormone levels and recent trends.
* A clinician dashboard to support multi-patient review and decision-making.

Core Functions

* Real-time and semi-continuous data acquisition
* PostgreSQL-backed storage and retrieval
* Visualisation of short- and long-term trends
* Modular architecture for cloud deployment

Aim

CortiSense aims to demonstrate a functional framework for hormone monitoring that integrates biosensing, secure data handling, and clinical usability. It is intended for both personal and clinical settings, with a focus on clarity, responsiveness, and medical relevance.
